# 基本语法
1. 符号说明
```autohotkey
#	Win
!	Alt
^	Control
+	Shift
&	连接两个按键(含鼠标按键)，合并成一个自定义热键
```
2. 常用命令
```autohotkey
Send 发送一段字符
SendInput 发送一段字符
MsgBox 弹出对话框
Run 运行一个软件或者打开网页
WinActivate 窗口激活
WinWaitActive 等待窗口激活
```
